INSPIRATION (For Devpost Story)
================================================================================

Privacy concerns with cloud-based AI coding assistants, API costs, 
offline development needs, and desire for complete control over code and data.

================================================================================
WHAT IT DOES (For Devpost Story)
================================================================================

CodeGenie is a local AI coding agent that provides intelligent code generation,
planning, error recovery, and project analysis - all running offline using 
Ollama. It breaks down complex tasks, writes context-aware code, detects and
fixes errors automatically, and learns from your coding style.

================================================================================
HOW WE BUILT IT (For Devpost Story)
================================================================================

Built with Python 3.9+ using Ollama for local LLM operations. Leveraged 
Tree-sitter for multi-language parsing, Pydantic v2 for data validation, 
Rich/Textual for beautiful terminal UI, and pytest for comprehensive testing.
Implemented custom safety features including sandboxed execution, command 
validation, and automatic backups.
